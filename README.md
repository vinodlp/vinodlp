### 🚀 Featured project

**[LLM Cost Guard](https://github.com/vinodlp/llm-cost-guard)**

FastAPI proxy that classifies every LLM request by complexity and routes it to the most cost-effective model tier automatically.

```
Simple requests   → Haiku   ($5/M tokens)
Moderate requests → Sonnet  ($15/M tokens)
Complex requests  → Opus    ($25/M tokens)
```

- **90% routing accuracy** on a 40-prompt diverse test set
- **54% cost reduction** vs sending all requests to Opus
- **~$49K/year** estimated saving at 100K requests/day
- Built with: Python · FastAPI · tiktoken · SQLite
